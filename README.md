📱 SIUPN App (Sistem Informasi UPN)

Aplikasi Android berbasis Kotlin yang dirancang untuk menampilkan informasi secara terstruktur melalui beberapa halaman seperti splash screen, homepage, daftar data, dan detail informasi.

---

📌 Fitur Utama

- Splash Screen (tampilan awal aplikasi)
- Homepage (halaman utama)
- List Data (menampilkan daftar informasi)
- Detail Data (menampilkan detail setiap item)

---

🛠️ Teknologi yang Digunakan

- Kotlin
- Android Studio
- RecyclerView
- XML Layout

---

📂 Struktur Project

com.siupn.app
├── splash/
├── home/
├── list/
├── detail/
├── adapter/
├── model/
└── data/

---

🚀 Cara Menjalankan Project

1. Clone repository ini
2. Buka menggunakan Android Studio
3. Tunggu proses build Gradle selesai
4. Jalankan aplikasi di emulator atau perangkat Android
